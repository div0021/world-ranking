## How To RUN

Go to main directory and run

```bash

# install node modules
$ npm install

# Run server
$ npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the website.
